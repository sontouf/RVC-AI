# UC-002: Forward cleaning while session active

## Name

**UC-002 — Forward cleaning while session active**  
청소 세션이 활성일 때 기본으로 **전진**하며 청소(및 걸레)를 수행한다.

## Actor

| Role | Actor |
|------|--------|
| Primary | RVC SW Controller (자율 제어) |
| Secondary | Sensors subsystem, Drive/cleaning actuators (블랙박스 인터페이스) |

## Pre-Requisites

- UC-001에 따라 세션 상태가 **Cleaning**이다.
- 전방·좌·우 **삼면 동시 막힘**이 아니다(해당 시 UC-004).
- **단방향 장애물 회피** 또는 **먼지 부스트**에 해당하는 조건이 **우선하지 않거나**, 처리 후 본 UC로 복귀한 상태다.

## Typical Courses of Events

1. 시스템이 센서로부터 **주변 인식**(전/좌/우 장애, 먼지 등)을 수신한다.
2. 시스템이 **전진·기본 청소 파워**로 주행·청소를 지시한다.
3. 장애·삼면막힘·먼지 부스트 조건이 없으면 **2**를 세션 중 반복한다(제어 루프).

## Alternative Courses of Events

- **A1. 먼지 감지(병행)**  
  UC-005에 따라 **일정 기간 파워 상향**이 적용되나, **전진 청소** 목표는 유지된다.

- **A2. 단방향 장애**  
  UC-003이 개입한 뒤 전진 청소가 재개되면 다시 본 UC의 전형적 흐름으로 돌아온다.

## Exceptional Courses of Events

- **E1. 삼면막힘**  
  UC-004로 전환한다.

- **E2. 세션 종료**  
  사용자 중지(UC-001) 또는 시스템 오류로 Cleaning이 끝나면 본 UC를 중단한다.

- **E3. 센서 신뢰 실패**  
  인식 불가·모순 데이터 시 안전 정지 또는 Idle — 정책은 NFR/구현.
